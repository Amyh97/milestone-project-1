why code institute?

I knew that quitting work in order to study was not a viable option for me, so I needed training to be remote and part-time. It was my dad who introduced me to The Code Institute, his company has hired some Code Institute graduates. The Industry Advisory Council (IAC) provides validation to Code Institute's courses so this, behind a university credit-rated diploma, looked like the best option. It was remote, with flexible working hours and a recognised qualification, so I chose The Code Institute to train with.
