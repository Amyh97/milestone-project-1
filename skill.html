why code institute?

I knew that quitting work in order to study wasn't a viable option for me, so however I trained needed to be remote and part-time. It was actually my dad that introduced me to Code Institute, he had heard of them as his company has hired some Code Institute graduates. The Industry Advisory Council (IAC) provides validation to Code Institute's courses, so with this behind a university credit-rated diploma looked like the best option. It was remote, flexible working hours and recognised, so I chose Code Institute to train with.
